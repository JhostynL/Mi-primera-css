Sección 1: Visita. La propiedad display es muy importante a la hora de posicionar los elementos en una página web ya que determina el espacio que va a ocupar un elemento. Existen 2 tipos básicos de display, el bloque(block) y en línea(inline)
Sección 2: Cada elemento o etiqueta de HTML tiene asignado por defecto un tipo de display, pero podemos cambiarlo. Las etiquetas con display en línea más usadas son la etiqueta, la etiqueta a (hiperenlace), etiqueta img y las etiquetas de formateo de palabras (strong, b, s, i, etc)
